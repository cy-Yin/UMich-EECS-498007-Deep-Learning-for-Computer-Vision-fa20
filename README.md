# UMich-EECS-498007-Deep-Learning-for-Computer-Vision-fa20

This repository contains the course materials for EECS 498-007/598-005 Deep Learning for Computer Vision, Fall 2020, at the University of Michigan, including 6 assignments and my solutions.

One can find the course website [here](https://web.eecs.umich.edu/~justincj/teaching/eecs498/FA2020/).

Lecture videos from the Fall 2019 offering are available on [YouTube](https://www.youtube.com/playlist?list=PL5-TkQAfAZFbzxjBHtzdVCWE0Zbhomg7r).

Assignments are designed to be completed on [Google Colab](https://colab.research.google.com/). To run the code, please upload all six assignment folders to your Google Drive, ensuring the path includes `2020FA`. Your Google Drive should be organized as follows:

```
My Drive/
    2020FA/
        A1
        A2
        A3
        A4
        A5
        A6
```

> [!NOTE]
> Lecture slides are not included in this repository due to their large file size. Please visit the [course website](https://web.eecs.umich.edu/~justincj/teaching/eecs498/FA2020/schedule.html) to access the lecture slides.
